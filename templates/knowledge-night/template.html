Pre-training: compress the internet
15T tokens of cleaned web text — language, facts, code — stored as lossy statistical compression in model parameters.
~15T tokens · FineWeb-scale
2
SFT: learn to be an assistant
Human experts demonstrate ideal responses. Model fine-tunes on examples, shifting from document predictor to conversation partner.
~100K examples · behavior alignment
3
RLHF: reinforce what works
RL rewards correct answers on verifiable tasks. Models discover chain-of-thought reasoning without being explicitly taught it.

Core Concepts
Four things to understand
before you use an LLM.
🔮
Token Prediction
Every output is a token-by-token continuation — "expensive autocomplete." Predicting well requires learning language, facts, code, and reasoning from training data.
Foundation
🧠
Context Window
The model has no persistent memory — only what's in the current context window. What's not in context doesn't exist, regardless of what was trained on.
Memory model
🔧
Tool Calling
Models emit structured commands — search, run code, read files — and results flow back into context. This loop is how LLMs become agents.
Action layer
⛓
Chain-of-Thought
Reasoning models learn to think step-by-step before answering — an emergent behavior discovered through RL on verifiable tasks, not explicitly programmed.
Reasoning strategy

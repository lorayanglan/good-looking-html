Use LLMs Right
Six
Rules That
Actually
Work
Parameters =
fuzzy memory.
Context =
working memory.
01 · Facts
Give It the Document
Don't rely on parameter memory. Paste the source or make it search. Context always beats recall.
02 · Compute
Let Code Do the Math
Arithmetic, counting, exact checks — hand these to a code interpreter. LLMs predict tokens, not correct digits.
03 · Reasoning
Give It Room to Think
More tokens = more computation. Hard problems need step-by-step room. Never force instant answers.
04 · Hallucination
Verify Before You Trust
The model generates what sounds right, not what is right. Any fact, number, or citation needs a human check.
05 · Tools
Offload What It Can't Hold
Search for facts. Run code for math. Query a DB for data. Tool calls swap fuzzy memory for exact answers.
06 · Mindset
Powerful Tool. Not Oracle.
Use it. Don't worship it. Check its work. You are still responsible for the final output.

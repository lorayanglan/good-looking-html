Netflix The Three Shifts · Trending Now
Karpathy's Software 3.0 — Top 3 Shifts
Apps That Will Disappear.
Skills That Will Dominate.
1
App Layer
Some apps will simply cease to exist
MenuGen could be an app — OCR, image API, auth, deployment. Or a user just hands a photo to a multimodal model and gets the output directly.
Ask: does this app still need to exist?
2
Execution Layer
Install .md skills, not .sh scripts
Shell scripts break across machines. A clear Markdown instruction set, handed to an LLM agent, is more robust — and readable by humans too.
English becomes executable code.
3
Knowledge Layer
Unstructured knowledge becomes computable
Classic code handles rows and JSON. LLMs handle articles, interviews, repos, and screenshots — compiling them into a structured, growing wiki.
From search to compile.

Sawtooth Intelligence
Model capability isn't uniform. Verifiable tasks — code, math, security tests — improve dramatically because feedback is automatic. Open-ended judgment lags behind.

Sensors, Actuators, Logic
Agent-native products expose three layers: Sensors (world state readable by agents), Actuators (safe executable actions), Logic (decisions split across humans, code, and models).
